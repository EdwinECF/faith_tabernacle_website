
# Church Website Project (In Progress)
[![CI](https://github.com/EdwinECF/faith_tabernacle_website/actions/workflows/ci.yml/badge.svg)](https://github.com/EdwinECF/faith_tabernacle_website/actions/workflows/ci.yml)
This project is a custom-built website for my church, created from scratch using HTML, CSS, and JavaScript.

## Purpose

The goal of this project is to move away from website builders like Wix and gain full control over design, functionality, and performance by writing everything manually.

At the same time, this project helps me strengthen my front-end development skills and better understand how real websites are built and structured.

## What I'm Working On

- Responsive layout and design
- Navigation and user experience
- Event sections and image sliders
- Clean and organized CSS structure
- JavaScript functionality for interactivity

## Why This Project Matters to Me

I’m currently transitioning into software development while working full-time, and projects like 
this help me build a strong foundation with real-world applications.

Instead of just following tutorials, I’m applying what I learn to something meaningful for my community.
